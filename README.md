# Aerocode - Sistema de Gestão de Produção de Aeronaves

Bem-vindo ao **Aerocode**, uma solução moderna de interface gráfica (GUI) desenvolvida para otimizar o acompanhamento do ciclo de vida de produção de aeronaves civis e militares.

## 🚀 Sobre o Projeto

Este projeto faz parte da Atividade de Avaliação Individual 2 (AV2) e representa a evolução de um sistema CLI para uma **Single Page Application (SPA)** robusta. A Aerocode foca na eficiência operacional, permitindo que engenheiros e operadores gerenciem desde o cadastro de componentes até a entrega final da aeronave.

## 🛠️ Tecnologias Utilizadas

- **React**: Biblioteca principal para construção da interface.
- **Vite**: Ferramenta de build para um desenvolvimento rápido e otimizado.
- **Tailwind CSS**: Framework de estilização utilitária para um design moderno e responsivo.
- **TypeScript**: Adição de tipagem estática para maior segurança e previsibilidade do código.
- **React Router**: Gerenciamento de rotas e navegação client-side.

## 📋 Funcionalidades Principais

- **Dashboard Central**: Painel com visão geral das aeronaves em linha de produção.
- **Gestão de Aeronaves**: Registro detalhado de modelos, códigos únicos, capacidade e alcance.
- **Controle de Inventário de Peças**: Rastreamento de peças nacionais e importadas com status em tempo real.
- **Fluxo de Etapas**: Gerenciamento sequencial de produção.
- **Sistema de Permissões**: Renderização condicional baseada no nível de acesso (Administrador, Engenheiro, Operador).
- **Simulação de Persistência**: Manipulação de dados estruturados baseada em mocks.

## ⚙️ Instalação e Execução

1. Clone o repositório
```
git clone https://github.com/JkDeltaz/AV2
```

2. Entre na pasta do projeto e na pasta frontend
```
cd aerocode/frontend
```

3. Instale as dependências
```
npm install
```

4. Inicie o servidor de desenvolvimento
```
npm run dev
```
